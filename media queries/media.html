I-Les différents types de media queries

All

All permet d'appliquer des règles à tous les écrans. C'est lui qui donne les règles par défaut.

Screen

Screen est utilisé pour appliquer des règles sur différents les appareils (ordinateur, téléphone, ...) qui possèdent un écran.

Print

Print est utilisé pour créer les pages "avant impressions".

Speech

Speech s'applique seulement aux liseurs d’écran qui rende le contenu d'une page web en page sonore.

II-Les caractéristiques

Width

width permet d'appliquer des règles à tous les écrans. C'est lui qui donne les règles par défaut.

min-width

min-width est utilisé lorsque l'on souhaite que les règles qui sont appliquées dans le média quéry soit utilisé jusqu'à ce que la taille de l'écran atteint la largeur minimum souhaitée(ex: min-width:800px)

max-width

max-width est utilisé lorsque l'on souhaite que les règles qui sont appliquées dans le média quéry soit utilisé jusqu'à ce que la taille de l'écran atteint la largeur maximum souhaitée (ex: max-width:1000px)

III-Les opérateurs logiques

Not

Not va inverser le résultat d'une requête uniquement sur celle où il est appliqué. Il ne peut pas inverser une autre caractéristique (only, and ...). Les règles de not vont donc s'appliquer à tous les appareils media sauf à celui qu'on lui indique.

Only

Only enlève les caractéristiques média lorsqu'ils ne peuvent pas etre pris en charge par un naviguateur. Cela s'applique seulement sur les anciennes versions des naviguateurs, les navigateurs moderne ne subiront aucun dégat.

And

And permet de séparer 2 conditions.

Virgule

La virgule permettent d'utiliser plusieurs requêtes
